1. Muammo → Yechim

Muammo

O‘zbekiston qishloq xo‘jaligida ko‘p fermerlar texnika va xizmatlarni topishda qiyinchilikka duch keladi. Asosiy og‘riq nuqtalari:

  • Traktor, kombayn, yer haydash yoki sug‘orish tizimi o‘rnatadigan mutaxassisni topish qiyin.
  • Xizmat narxlari bir xil emas — oshirilgan yoki adolatsiz narxlar mavjud.
  • Ishonchli pudratchini ajratish qiyin: soxta profillar, kelmaslik, sifatsiz xizmat.
  • Fermerlar telefon orqali tanish-bilish orqali izlaydi — vaqt va kafolat yo'q.
  • AgroBank subsidiya/kredit doirasidagi xizmatlar nazoratsiz bo‘lishi mumkin.

Yechim — AgroService (qisqacha)

AI asosida ishlaydigan agro-xizmatlar platformasi: fermerlar va texnika/xizmat egalari bir joyda bog‘lanadi. Platforma quyidagi asosiy funksiyalarni taklif etadi:

  • AI Matching — foydalanuvchi talabi (lokatsiya, maydon, ekin, vaqt) asosida eng mos 3–5 provayder.
  • AI Narx Baholash — bozor ma'lumotlari asosida adolatli narx diapazoni (min–avg–max).
  • Ishonchlilik Reytingi (Risk Detection) — profil va sharhlar tahlili orqali xavf balli.
  • Avtomatik Jadval (Smart Scheduling) — provayderning bo‘sh va band vaqtlariga qarab ideal vaqtni tavsiya qiladi.

Afzalliklar: tez, adolatli, ishonchli xizmat — fermerlar uchun; yangi mijozlar va aniq jadval — provayderlar uchun; shaffoflik — AgroBank uchun.

4. Yo'l xaritasi: Idea → Prototype → MVP → Launched

Idea (Hozir)

Prototype (2-4 kun)

MVP (1–2 hafta)

Launched (after tests)

Beta test (3–5 fermer), xavfsizlik, mobil optimizatsiya, AgroBank integratsiyasi va rasmiy ishga tushirish.

5. Yechimni qanday amalga oshirishni rejalashtirayapmiz

Bosqichlar (an’anaviy):

  1. Arxitektura va reja (2 kun): DB sxemasi, API'lar, AI modul joylashuvi, Figma prototip.
  2. Frontend prototip (2-3 kun): React/Next.js, Tailwind, responsive dizayn.
  3. Backend va API (3-5 kun): FastAPI, PostgreSQL, auth (JWT), CRUD endpointlar.
  4. AI modullar integratsiyasi (3-7 kun): Rule-based matching, price estimator (statistical), risk detection (rule-based → ML).
  5. Sinov va birlashtirish (4-5 kun): End-to-end test, demo tayyorlash.
  6. Beta testi va feedback: 3–5 fermer bilan sinov, user feedbackga ko‘ra sozlash.
  7. Launch: Deploy (Vercel/Netlify + Railway), marketing va region bo‘yicha kengaytirish.

Texnologiyalar va AI vositalari

  • Backend: Python FastAPI, PostgreSQL, SQLAlchemy
  • AI & Data: Pandas, NumPy, scikit-learn, rule-based engine. Chatbot/UX yordam uchun GPT API (OpenAI) — foydalanuvchi savollariga tez yordam.
  • Infra: GitHub actions (CI), Vercel/Netlify (frontend), Railway/Render (backend)

Hakaton uchun minimal AI qarorlari (qisqacha)
